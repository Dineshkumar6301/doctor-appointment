🩺 Doctor Appointment Booking System
This is a Django-based web application that allows patients to book appointments with doctors. The platform supports user registration/login for both doctors and patients, profile management, appointment scheduling, and availability tracking.

🚀 Features
👨‍⚕️ Doctor & Patient Registration/Login
📅 Appointment Booking & Management
🕒 Doctor Schedule Availability
📄 Doctor Listings with Reviews
🔒 Secure Login with Password Reset
📬 Contact and Messaging System (optional)
🛠 Tech Stack
Backend: Django, SQLite
Frontend: HTML, CSS, Bootstrap, JavaScript
Tools: Git, GitHub, VS Code
